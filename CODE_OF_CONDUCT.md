# Code of Conduct

This project follows the [Contributor Covenant 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

## Short version

- Be respectful. Disagree about ideas, not people.
- Assume good faith. Most "wrong" PRs are misunderstandings, not malice.
- No harassment, no slurs, no personal attacks. We're here for the code.
- If you see a violation, report it to the maintainer privately (see repo profile for contact).

## Scope

Applies to all project spaces — issues, PRs, discussions — and to representations of the project elsewhere (talks, blog posts, social media linking back to the repo).

## Enforcement

The maintainers will read reports and act. Possible actions: private warning, public warning, temporary ban, permanent ban. The maintainer's decision is final; if you disagree, fork.
